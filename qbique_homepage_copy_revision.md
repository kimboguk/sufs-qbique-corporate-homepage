# Qbique 홈페이지 카피 개선안

## 작성 배경

기존 홈페이지 초안은 "포트폴리오 최적화가 단순한 자산 배분이 아니다"라는 기능적 서술로 시작하여, 비즈니스가 성립해야 할 근본적인 이유를 충분히 전달하지 못하고 있었음. 아래는 "왜 이 플랫폼이 존재해야 하는가"를 선행한 뒤, 기술적 차별점으로 이어지는 구조로 재구성한 카피 개선안임.

---

## 1. 히어로 섹션 (개선안)

### 헤드라인

**기관의 역량을, 모든 투자자에게**

### 서브 헤드라인

*지표를 보여주는 것과 투자를 돕는 것은 다릅니다*

### 본문

기존 시스템은 시장 데이터 분석 도구를 제공할 뿐이고, ETF와 같은 상품은 누군가의 판단을 따르게 합니다. Qbique는 그 사이에서 — 시장의 리스크 구조를 가공하여 투자자가 직접 최적화된 포트폴리오에 체계적으로 도달할 수 있는 경로를 제공합니다.

---

## 2. 문제 제기 섹션 (신규)

> 기존 "금융기관이 직면한 운용의 현실" 섹션을 대체

### 섹션 타이틀

**투자는 대중화되었지만, 도구는 여전히 트레이딩에 머물러 있습니다**

### 본문

개인 투자자에게도 분석 도구는 제공됩니다. 그러나 대부분은 개별 종목 중심 분석과 트레이딩 전략 구현에 치중되어 있어서 투자자를 더 빈번한 거래로 이끄는 구조에 가깝고, 포트폴리오 수준의 리스크를 다루지 않습니다. 아무리 정교한 실시간 지표가 제공되더라도, 대부분은 시장 가격에 후행할 뿐만 아니라 노이즈 속에서 신뢰할 수 있는 신호를 포착하기 어려우며, 거래 비용까지 고려하면 단기 매매로 지속적인 우위를 확보하기는 더욱 어렵습니다.

이를 인식한 투자자들은 ETF와 같은 상품으로 향하지만, 그 순간 투자는 완전히 수동적인 활동이 됩니다. 능동적으로 시장에 참여하되 포트폴리오 관점에서 체계적으로 운영할 수 있는 도구는 사실상 부재합니다.

Qbique는 직관적인 사용성을 바탕으로 이 간극을 채웁니다.

## 3. 이후 섹션 구조 (권장)

히어로와 문제 제기 이후, 기존 섹션들은 다음 순서로 재배치를 권장함:

### 솔루션 개요 (기존 "Qbique 포트폴리오 최적화 플랫폼" 섹션)

- 통합성, 유연성, 기술 선도, 사용 편의성
- 기존 내용 유지 가능하나, 문제 제기 섹션에서 이어지는 맥락에 맞게 톤 조정 필요

### 핵심 가치 (신규 — 솔루션 직후, 기능/기술 이전에 배치)

아래 순서로 제시하여, 구체적 메커니즘에서 비전으로 확장되는 흐름을 구성:

1. 투자 의사결정을 이끄는 활용 가능한 정보 가공
2. 리스크 관리의 체계화
3. 직관적 사용성
4. 기관 수준 분석에 대한 접근성 제고

### 주요 기능 (기존 유지)

- 유연한 전략 구현 환경
- 고급 최적화 엔진
- 문제 정의 및 리포팅
- ChatDevOps

### 기술적 차별점 (기존 유지)

- 휴리스틱 + 임베디드 최적화
- 양자-고전 하이브리드 최적화
- 머신러닝 기반 수익률 전망

### 고객별 가치 (조정 완료)

- 개인 투자자 / 기관 관리자: 기관 수준 분석 접근성, 직관적 사용성, 능동적 투자
- 기관 투자자 / 개인 전문가: 차별화된 최적화 알고리즘, 맞춤 모델링 환경, 운용 효율화
- 데이터 / 기술 파트너: 시장 데이터 밸류체인, 기술 생태계 협업

## 4. 핵심 키워드 정리

논의 과정에서 도출된 Qbique의 핵심 가치 표현 (제시 순서):

| 키워드 | 설명 |
|--------|------|
| 투자 의사결정을 이끄는 활용 가능한 정보 가공 | 리스크 구조와 수익률 전망을 포트폴리오 구성에 직접 반영 |
| 리스크 관리의 체계화 | 감정적 매매를 지양하고 체계적 포트폴리오 운영 지원 |
| 직관적 사용성 | 복잡한 투자 의사결정 과정을 쉬운 사용 경험으로 전환 |
| 기관 수준 분석에 대한 접근성 제고 | 정보 처리와 리스크 관리 역량을 모든 투자자에게 |
| 트레이딩 도구와 패시브 상품 사이의 간극 | Qbique의 시장 포지셔닝 — 단순 지표 나열도, 수동적 상품도 아닌 능동적 포트폴리오 운영 영역 |

## 5. 참고: 서사 구조 요약

```
히어로          → 선언: 기관의 역량을 모든 투자자에게
                  포지셔닝: 분석 도구와 ETF 사이의 빈 공간

문제 제기        → 근거: 왜 이 플랫폼이 대중에게 열려야 하는가
                  기존 도구는 트레이딩 중심 → 포트폴리오 사고 부재
                  ETF는 수동적 → 능동적 운영 영역의 공백

솔루션          → 개요: 통합성, 유연성, 기술 선도, 사용 편의성

핵심 가치        → 왜 가치 있는가: 정보 가공 → 리스크 체계화 → 사용성 → 접근성 제고

기능/기술        → 방법: 구체적으로 어떻게 풀고 있는가
                  기술적 신뢰성의 근거

고객별 가치      → 적용: 각 고객군에게 어떤 가치를 주는가
                  개인/관리자: 접근성과 사용성
                  기관/전문가: 차별화된 알고리즘과 모델링 환경
                  파트너: 데이터 밸류체인과 생태계 협업
```
